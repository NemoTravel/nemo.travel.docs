### Cost

Contains information about the amount and currency of the cost.

#### Formar description 

-   **Amount** - amount.
-   **Currency** - currency code.

#### Example

The element indicates the base cost for one passenger.

```xml
<BaseFare>
  <a:Amount>6319</a:Amount>
  <a:Currency>KZT</a:Currency>
</BaseFare>
```
