### Cost

Contains information about the amount and currency.

#### Formar description 

-   **Amount** - The amount.
-   **Currency** - The currency code.

#### Example

The element indicates the base cost for one passenger.

```xml
<BaseFare>
  <a:Amount>6319</a:Amount>
  <a:Currency>KZT</a:Currency>
</BaseFare>
```
