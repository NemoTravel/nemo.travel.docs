### Стоимость

Содержит информацию о сумме и валюте стоимости.

#### Описание формата

- **Amount** — сумма.
- **Currency** — код валюты.

#### Пример

Элемент указывает базовую стоимость для одного пассажира.

```xml
<BaseFare>
  <a:Amount>6319</a:Amount>
  <a:Currency>KZT</a:Currency>
</BaseFare>
```
