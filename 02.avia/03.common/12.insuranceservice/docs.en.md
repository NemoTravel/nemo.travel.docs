### InsuranceService

The insurance.

-   **Vendor** - insurance services vendor. Data type - enumeration, possible values:
    -   AlphaInsurance
-   **InsuranceLevel** - level/program of insurance. Data type - string.
-   **DocumentURL** - reference to the document with insurance. Data type - string.
