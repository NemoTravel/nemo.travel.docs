### HostCommand

Выполнение терминальной команды в GDS.

#### Запрос

##### Описание формата

-   **Source** - ID пакета реквизитов (источника), под которыми требуется выполнить команду. Тип данных - целое 32-битное число. (обязательное поле)
-   **SessionID** - ID сессии, в рамках которой будет выполнена команда. Тип данных - строка.
-   **Command** - Выполняемая команда. Тип данных - строка. (обязательное поле)

##### Примеры

```xml
<Source>1</Source>
<SessionID>9c47e9d3-6103-4495-bf22-f56379e5d2b1+0</SessionID>
<Command>AAA</Command>
```

#### Ответ

##### Описание формата

-   **Response** - Результат выполнения терминальной команды. Тип данных - строка.

##### Примеры

```xml
<Response>ФАААА</Response>
```
